# 复合类型
## 字符串
>> 字符串和字符数组不一样
```c++
    char dog[6] = {'d', 'e', 'u',' ', 'v', 'p'};//字符数组
    char cat[8] = {'f', 'a', 't', 'e', 's', 's', 'a', '\0'}; //字符串 char cat[8] = "fatessa";
    注意 ‘\0’ 的位置
```
>> 在确定存储字符所需的最短数组时，要将结尾的空字符计算在内。  
>> 输入 cin 的方法 get() & getline() 有失效位（failbit）
----
## string 类 （位于std名称空间中）
> char 数组视为一组存储一个字符串的 char 存储单元；  
> string 类变量是一个表示字符串的实体。
