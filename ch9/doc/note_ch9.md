# 存储
## 静态变量
>- 所有静态变量创建之后都被零初始化（而不管程序员是否显示初始化它）。
>- 接下来如果使用了常量表达式初始化了变量，且编译器仅根据当前文件内容（包括被包含的头文件）就可以计算表达式，编译器将执行常量表达式初始化（必要时候将进行简单计算，若没有足够信息，变量将被动态初始化，如函数需要等到链接且程序执行的时候）  
>- extern 关键字告诉编译器，其他地方已经定义了 xxx, 去其他地方找。
